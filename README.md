# NovaFlow AI

NovaFlow AI is a modern AI-themed frontend built with Next.js. It showcases a concept interface for building AI agents, automating workflows, and integrating tools through a clean and responsive design.

## Tech Stack
Next.js  
React  
TypeScript  
Tailwind CSS  

## Getting Started

Clone the repository:

git clone https://github.com/shivammane2007/NovaFlow_AI.git

Install dependencies:

npm install

Run the development server:

npm run dev

Open http://localhost:3000 to view the project.

## Purpose

This project demonstrates a modern frontend architecture and UI design for an AI workflow platform.
